# Customer Churn Analysis & Retention Insights Dashboard

This project focuses on analyzing customer churn behavior and identifying key factors affecting customer retention using Power BI.

 

##  Project Overview
The objective of this project is to analyze customer data and uncover patterns that influence churn.  
An interactive dashboard is created to visualize trends and support data-driven decision-making.

 

##  Dashboard Features
- Total Customers and Churned Customers overview  
- Churn analysis by Internet Service Type  
- Customer distribution by Gender  
- Churn distribution by Payment Method  
- Customer distribution by Contract Type  
- Impact of Monthly Charges on Churn  
- Customer churn distribution based on Tenure  

 

##  Key Insights
- Higher churn observed in month-to-month contracts  
- Fiber optic users show higher churn rates  
- Customers with higher monthly charges tend to churn more  
- Payment methods influence customer retention behavior  
- Minimal churn difference across genders  

 

##  Tools Used
- Power BI  
- Data Cleaning  
- Data Visualization  
- Data Analysis
